## 四、红黑方胜率变化

| 对手 | 执红 D-06 | 执红 D-07 | 执黑 D-06 | 执黑 D-07 |
|------|-----------|-----------|-----------|-----------|
| random | 199/200 = 99.50% | 199/200 = 99.50% | 199/200 = 99.50% | 199/200 = 99.50% |
| greedy | 181/200 = 90.50% | 180/200 = 90.00% | 172/200 = 86.00% | 183/200 = 91.50% |
| tactical | 75/100 = 75.00% | 79/100 = 79.00% | 78/100 = 78.00% | 87/100 = 87.00% |

**分析**：

- **Random**：红黑双方均无变化，稳定 99.50%。
- **Greedy**：整体 +2.5pp 的改善主要来自执黑侧（86.00% → 91.50%，+5.5pp），执红侧基本持平（90.50% → 90.00%）。这提示 C-19 可能改善了后手方的搜索质量。
- **Tactical**：红黑两侧均有提升（红 +4pp，黑 +9pp），执黑侧变化更大。但由于分侧样本仅 100 局，波动范围更大。

**重要**：分侧样本量（每对手每侧 100–200 局）远小于总体比较（200–400 局），**不能据此宣称统计显著**。这些数字仅作为方向性参考，供 C 团队后续优化时判断是应优先加强先手还是后手。

## 六、综合结论

### 选择结论#2：性能改善、胜率基本持平

**依据**：
1. avgDepth 对所有对手均有改善（尤其 tactical +64%）
2. timeoutRate 对所有对手大幅下降（random -15pp, greedy -21pp, tactical -33pp）
3. avgNodesPerMove 增加约 80-95%，搜索效率明显提升
4. 三个对手的胜率变化均在 95% 置信区间内，**不能证明胜率发生统计显著变化**
5. Random 稳定 99.5% > 95%，Greedy 90.75% > 80%，Tactical 83.0% > 80%
6. 没有出现回退——无任何对手的胜率或性能指标恶化

### C-19 归类判断

C-19 属于**明显改善**（按性能指标判断）同时**胜率基本持平**（按统计检验判断）。

搜索效率的提升是真实的、一致的、量化的，但要在 200-400 局样本中检测 ±2-6% 的胜率差异需要更大样本。当前数据支持"C-19 让搜索更高效，在相同时间预算内探索更多节点，但胜率层面的效果尚需更大规模验证"。
